# E-Commerce App

Full Stack MERN E-Commerce Web Application.

## Features
- User Register/Login
- Protected Routes
- Product Search
- Add to Cart
- Quantity Management
- Checkout Form
- Order Placement
- Order History

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- FakeStore API

## Run Locally

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start
